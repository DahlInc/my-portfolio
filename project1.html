Immersive Three.js 3D Environment

Introduction

This project is a dynamic 3D interactive portfolio website that leverages the powerful Three.js library for real-time 3D rendering, combined with various web technologies such as GSAP for smooth animations, and TWEEN.js for easing control. The website is styled as an interactive Ramen Shop, where users can explore a virtual ramen shop filled with various objects representing projects, features, and interactive elements. This project demonstrates a deep integration of 3D modeling, web interactivity, and advanced visual effects like bloom, lighting, and animations.

Design

The centerpiece of the project is a fully modeled 3D ramen shop created in Blender and exported as a GLB file. This model serves as the primary environment of the website, providing an immersive experience for the user. Within the 3D scene, certain objects are clickable, allowing users to interact with them in various ways. For example, project placeholders (represented by images) are placed throughout the scene, and when clicked, the camera zooms in to give the user a closer view of the project. This functionality is built using raycasting in Three.js to detect object clicks and GSAP for smooth camera transitions.

Implementation

The interactive scene features a continuously spinning fan animated with Three.js to enhance realism. Advanced lighting techniques, including point, ambient, and directional lights, create a vibrant atmosphere, while bloom effects on elements like lamp posts add a cyberpunk glow using Three.js’s UnrealBloomPass. Smooth user interactions are managed with OrbitControls, allowing camera rotation, zoom, and panning, while locking focus on specific objects when needed. The portfolio showcases dynamic, clickable PNG project images that zoom into detailed views upon interaction. Additionally, background music loaded via Three.js’s AudioLoader immerses users further into the cyberpunk aesthetic.

Purpose and Use Case:

This project serves as a portfolio showcase for interactive web development and 3D modeling skills. The combination of Blender for 3D asset creation and Three.js for rendering highlights the developer’s ability to merge design and functionality seamlessly. The interactive nature of the portfolio demonstrates proficiency in user experience (UX) design, where intuitive control and visual effects are balanced for a smooth, immersive experience. The use of advanced technologies like GLTF model loading, real-time lighting, and audio integration adds depth and realism to the scene. In summary, this project is not just a portfolio showcase but an example of how 3D environments can enhance the presentation of creative work in web development. By merging interactive elements with smooth transitions, realistic lighting, and audio, the project elevates the standard web portfolio to an engaging and immersive experience
